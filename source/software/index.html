Fog Cloud Software

The Fog Cloud project aims to produce the following as Free/Open Source software.

This software will be published on the Fog Cloud Github Page.

Fog Sync
Fog Sync will allow you to securely synchronize your data across multiple computers.
Fog Sync for Android
Fog Sync for Android will extend Fog Sync to support Android phones and tablets.
Fog Mail
Fog Mail will provide a mechanism to make your electronic mail more secure.
